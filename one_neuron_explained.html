🎯 "Each" = Each Individual Neuron

Layer 1 has 128 neurons. EVERY neuron gets the SAME 784 pixels as input. But each neuron has its OWN 784 weights — so each neuron looks for a DIFFERENT pattern.

📐 Inside ONE Neuron: The Full Calculation

// Neuron #1 receives ALL 784 pixels as input // It has its OWN 784 weights (w₁ to w₇₈₄) and 1 bias (b) Input (same for ALL 128 neurons): pixel₁ pixel₂ pixel₃ ... pixel₇₈₄ 0.0 0.0 0.12 ... 0.0 ← flattened 28×28 image Weights (UNIQUE to Neuron #1): w₁ w₂ w₃ ... w₇₈₄ 0.02 -0.15 0.78 ... -0.03 ← what this neuron "cares about" // The computation (same for every neuron): z = (w₁×p₁ + w₂×p₂ + w₃×p₃ + ... + w₇₈₄×p₇₈₄) + bias // Example with real numbers: z = (0.02×0.0) + (-0.15×0.0) + (0.78×0.12) + ... + (-0.03×0.0) + 0.05 = 0.0 + 0.0 + 0.094 + ... + 0.0 + 0.05 = 0.52 ← Neuron #1's output (one single number!) // Then ReLU activation: if z > 0, keep it; else output 0 output = max(0, 0.52) = 0.52 ← passed to Layer 2

⚖️ What the Bias Does (The +1)

// Without bias: neuron can ONLY fire when pixels match its weights // With bias: neuron can fire even when input is all zeros z = (w₁×p₁ + ... + w₇₈₄×p₇₈₄) + bias ↑ This one extra number shifts the threshold // Think of bias as the neuron's "default excitement level": bias = +0.5 → neuron fires MORE easily (lower threshold) bias = -0.5 → neuron fires LESS easily (higher threshold) bias = 0.0 → neuron fires only when weights match perfectly // Bias is learned during training, just like weights // Every neuron has exactly ONE bias number
