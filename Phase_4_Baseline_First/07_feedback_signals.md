# 4.7 — Real-World Feedback Signals

| Signal | What It Measures |
|--------|-----------------|
| Adoption Rate | % of agents who look at ML estimate |
| Agreement Rate | % agents within 10% of ML estimate |
| Time Saved | Minutes per listing before vs after ML |
| Error Patterns | Which segments get bad predictions? |
| Qualitative | Agent comments, complaints, praise |
| Feedback Loop | Predicted → Actual sale price → retraining data |
